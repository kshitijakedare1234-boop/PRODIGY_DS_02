## Task 02 - Data Science Internship at Prodigy Infotech

### Objective
Perform data cleaning and exploratory data analysis (EDA) on a dataset to understand patterns and relationships between variables.

### Dataset
Titanic Dataset

### Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn

### Steps Performed
1. Loaded the Titanic dataset using pandas.
2. Checked for missing values in the dataset.
3. Filled missing values in the Age column.
4. Visualized survival count using bar charts.
5. Analyzed survival rate by gender.
6. Plotted age distribution using histogram.

### Output
- Survival Count Chart
- Survival by Gender Chart
- Age Distribution Histogram

### Conclusion
The analysis shows how factors like gender and age affected survival rates in the Titanic dataset.
